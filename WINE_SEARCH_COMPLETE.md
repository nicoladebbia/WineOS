## 🚀 HOW TO USE

### **Adding a Wine - The Easy Way:**

1. **Open Add Wine Screen**
2. **Tap "Search Wine Database"** (big wine-red button at top)
3. **Search for wine:**
   - Type "Brunello" → See Brunello di Montalcino
   - Type "Barolo" → See Barolo options
   - Type "Franciacorta" → See Ca' del Bosco!
4. **Select wine** from results
5. **Choose producer** (or enter custom)
6. **Enter year** (e.g., 2015)
7. **Form auto-fills:**
   - ✅ Name
   - ✅ Year
   - ✅ Type
   - ✅ Country
   - ✅ Region
   - ✅ Grape Variety
8. **Fill in your details:**
   - Supplier
   - Prices
   - Quantities
9. **Save!**

**Time saved: 2-3 minutes per wine!**
