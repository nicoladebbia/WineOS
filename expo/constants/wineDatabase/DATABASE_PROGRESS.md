## 💡 IMPLEMENTATION NOTES

Each wine entry includes:
- Unique ID
- Wine name
- Type (Red/White/Rosé/Sparkling/Dessert)
- Country
- Region
- Sub-region
- Grape variety/varieties
- Classification (DOCG, DOC, IGT, etc.)
- Minimum vintage year
- Average price (in €)
- Producer(s)
- Aliases for search

All files are TypeScript with proper type safety using the `WineTemplate` interface.
